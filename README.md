# Projeto PHP - Estudos

Projeto desenvolvido para praticar lógica de programação, sintaxe da linguagem PHP e conceitos básicos de desenvolvimento back-end.

## Sobre o projeto

Este projeto foi criado com o objetivo de aprender PHP na prática, explorando recursos essenciais da linguagem, como:

- variáveis e tipos de dados
- estruturas condicionais
- laços de repetição
- funções
- arrays
- entrada de dados
- manipulação de strings
- lógica de programação

A ideia é usar este repositório como um espaço de estudo e evolução, com exercícios, testes e pequenos desafios desenvolvidos ao longo do aprendizado.

## Tecnologias utilizadas

- PHP
- HTML
- CSS
- Git e GitHub

## Objetivo

O principal objetivo deste projeto é fortalecer a base em PHP por meio da prática, entendendo como a linguagem funciona e como ela pode ser utilizada no desenvolvimento de aplicações web.

## Funcionalidades

Este projeto pode conter exemplos como:

- cálculos matemáticos
- manipulação de listas e arrays
- exercícios de lógica
- funções personalizadas
- exibição de dados na tela
- tratamento básico de entradas
